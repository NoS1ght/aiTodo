# TaskTodo - 学生智能任务管理

## 启动方式

> **双击 启动.cmd 即可**，会自动打开浏览器进入应用。

### 手动启动（备用）

`powershell
.\start.ps1
`

### 启动后

| 服务 | 地址 |
|---|---|
| 前端页面 | http://localhost:8766 |
| 后端 API | http://localhost:8765 |

关闭：在命令行窗口中按 **Enter**，或直接关闭终端。

## 功能说明

### 1. 配置 AI（可选）

点击右上角 **⚙️ 太阳图标**，填写：

| 字段 | 说明 | 示例 |
|---|---|---|
| API Base URL | API 地址 | https://api.deepseek.com |
| API Key | 你的密钥 | sk-xxx... |
| 模型 | 模型名称 | deepseek-chat |

> 配置会自动保存到本地 llm_config.json，下次打开无需重复输入。

### 2. 添加任务

在 **「全部事件」** 标签页下：

**手动添加**
- 点击弹窗右上角 **「＋」按钮** → 填写标题、描述、优先级、截止时间 → 确认

**AI 提取（消耗 Token）**
- 在文本框粘贴一段文字 → 点 **「AI 提取」**
- 例如：*"明天下午3点交数学作业，周五前复习英语"*
- AI 会自动解析出：标题、截止时间、预估时长

**图片提取（需多模态模型）**
- 截图/拍照 → 点击 **「上传图片」**
- 需配置支持多模态的 API（如 gpt-4o、deepseek-vision）
- AI 会识别图中文字并提取任务

### 3. 任务安排（不消耗 Token）

点击 **「执行调度」** 或切换 **「每日」** 标签查看今日计划。

- 调度算法纯本地运行，**无需调用 API**
- 自动根据**优先级 + 截止时间紧迫度**计算紧急分数
- 逾期任务自动排在最前
- 每天按 8 小时容量自动分配
- 昨天未完成的自动滚动到今天

### 4. Token 消耗总结

| 操作 | 消耗 Token |
|---|---|
| AI 提取事件（文字/图片） | ✅ 消耗 |
| 事件自动调度安排 | ❌ 不消耗 |
| 手动添加/编辑/删除 | ❌ 不消耗 |
| 每日计划视图 | ❌ 不消耗 |

## 技术栈

| 层 | 技术 |
|---|---|
| 后端 | Python + FastAPI |
| 前端 | 原生 HTML / CSS / JavaScript |
| 数据库 | SQLite（文件：	asktodo.db） |
| AI | OpenAI 兼容 API（默认 DeepSeek） |
| 调度 | 贪心算法（纯本地，零 AI 依赖） |
